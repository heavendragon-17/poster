01. Đặt vấn đề
•ĐỘNG LỰC SUY GIẢM: duy trì động lực học Toán ở học sinh tiểu học là thách thức lớn — thế hệ lớn lên cùng smartphone và game khiến phương pháp truyền thống kém hấp dẫn.
•GAMIFICATION: học sinh học tốt hơn trong môi trường có phần thưởng, cạnh tranh lành mạnh và phản hồi tức thì.
•CƠ HỘI TỪ AI: mô hình ngôn ngữ lớn (LLM) cho phép trợ lý học tập giải thích từng bước, gợi ý theo ngữ cảnh và độ khó phù hợp từng học sinh.
•GIẢI PHÁP — MELA: ứng dụng di động học Toán kết hợp nội dung học thuật có cấu trúc + cơ chế gamification + trợ lý AI Gemini.

06. Kết luận
Giá trị mang lại
•ỨNG DỤNG HOÀN CHỈNH: Flutter đa nền tảng iOS/Android, Clean Architecture, 14+ tính năng chính, cache offline (Sembast).
•BACKEND VỮNG CHẮC: NestJS 30+ module, PostgreSQL 41 model Prisma, Redis, WebSocket — API P95 < 200ms.
•VÒNG LẶP GAMIFICATION đầy đủ: XP · Token · Crown · Heart · Streak · Badge · Gacha · Shop · Leaderboard — phần thưởng idempotent nhờ Redis distributed lock.
•TRỢ LÝ AI GEMINI: giải toán từng bước, gợi ý và đáp án theo ngữ cảnh lớp học.
•KIỂM THỬ: 86 test case pass (39 backend unit+E2E, 27 Flutter, 20 manual trên thiết bị thực); triển khai UAT + Production trên VPS với Docker, CI/CD GitHub Actions.
Hướng phát triển
•NGẮN HẠN (3–6 th): skill-based matchmaking (ELO), AI streaming response, hoàn thiện CMS cho giáo viên.
•TRUNG HẠN (6–12 th): AI tự sinh câu hỏi theo chủ đề/độ khó, tính năng xã hội (bạn bè, guild), tích hợp lớp học cho giáo viên.
•DÀI HẠN (12+ th): mở rộng sang Tiếng Anh, Khoa học; multi-tenant B2B cho trường học và trung tâm.
•LEARNING ANALYTICS: module phân tích dữ liệu học tập sâu hơn cho giáo viên và phụ huynh.
